# 🇦🇴 Afri Language (Versão Alpha)

O **Afri** é uma linguagem de programação interpretada, de código aberto, idealizada, fundada e desenvolvida por **José Carlos Pipa**. 

Construída sobre o ecossistema Python, o seu principal propósito é ser uma ferramenta educacional e conceitual, trazendo uma sintaxe totalmente baseada na língua portuguesa para facilitar o aprendizado de lógica de programação e a estrutura de compiladores para a comunidade tecnológica de Angola e do mundo.

O motor do Afri possui um pipeline clássico composto por um **Lexer** (Analisador Léxico), **Parser** (Analisador Sintático que gera uma AST) e um **Interpretador** dinâmico com gestão de memória.
